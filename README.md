# Sports Buddy

Sports Buddy is a web-based sports networking application that helps users
connect with nearby players and sports events.

## Tech Stack
- HTML, CSS, JavaScript
- Firebase (Auth + Firestore)

## Features
- User & Admin login
- Sports event management
- Location-based discovery
- Logging for all actions

## How to Run
1. Clone repo
2. Configure Firebase
3. Open index.html
